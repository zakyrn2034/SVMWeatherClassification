# Klasifikasi kondisi cuaca menggunakan SVM
Model ini menggunakan pembacaan nilai-nilai di lingkungan sekitar (suhu, kelembaban, dan persentase tudung awan) untuk memprediksi kondisi cuaca menjadi dua macam, hujan atau tidak hujan.
## Cara penggunaan
Jalankan kode pada notebook sesuai urutan. Notebook terbagi menjadi beberapa bagian, sesuai dengan fungsi kode pada bagian tersebut.
